## 1) Find tailing zero (factorial)
 
- **Path**: `1_find_tailing_zero/main.py`
- **Description**: หาจำนวนเลข 0 ที่อยู่ติดกันหลังสุดของค่า factorial
 
### Run
 
```bat
python 1_find_tailing_zero\main.py
```
 
### Input
 
- [Example 1]
input = 7
output = 1

- [Example 2]
input = -10
output = number can not be negative
